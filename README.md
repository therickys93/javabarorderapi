## installazione con Gradle

aggiungere repository jitpack nel file root build.gradle

```
allprojects {
	repositories {
		...
		maven { url 'https://jitpack.io' }
	}
}
```

aggiungere nel file del progetto

```
dependencies {
        compile 'com.github.therickys93:javabarorderapi:1.1.0'
}

```
